Introduction to Html

HTML (Hypertext Markup Language) is the code used to structure a webpage and it's content.

PS; it was invented by Physcist Tim Berners-Lee, the first version was produced in 1993.

So what is Html?

Html is a markup language that defines the structure of the content.

It consist of a series of elements which is used to close or wrap different parts of the content to make it appear or act a certain way. For example, if we wanted a line to stand alone, we could specify it in a paragraph by enclosing it in a paragraph tag;

Anatomy of Html

The main parts of Html are as follows;

  1. The opening tag; this is the name of the element (for example p) wrapped in open and close angle bracket. This states where begins or start to take effect
  2. The closing tag;
    3.  This is the same with the opening tag except that it includes a forward slash before the element name. This states where the element ends - In this case where the paragraph ends.
  3. Others

Summary

Html language is a markup text used to describe structure and content of webpage. It states the structure of the page. A basic Html elements are listed below;

  • Head
  • Body
  • paragraph
  • list
